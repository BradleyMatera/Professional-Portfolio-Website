## üöÄ Running the Demo

### Development Mode
```bash
npm start
```
Opens at [http://localhost:3000](http://localhost:3000).  

### Production Build
```bash
npm run build
```
Builds optimized files in `/build`.  

### Local Serving
```bash
npm install -g serve
serve -s build
```
Opens at [http://localhost:5000](http://localhost:5000).
